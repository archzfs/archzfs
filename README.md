# Zettabyte File System for Arch Linux

[![GitHub Actions Workflow Status](https://img.shields.io/github/actions/workflow/status/archzfs/archzfs/release.yml?branch=master&label=release)](https://github.com/archzfs/archzfs/actions/workflows/release.yml?query=branch%3Amaster++)
[![GitHub Release Date](https://img.shields.io/github/release-date/archzfs/archzfs)](https://github.com/archzfs/archzfs/releases/tag/experimental)

Welcome to the ArchZFS project. This repo contains everything used to deploy ZFS to Arch Linux.

## Overview

Take a look at the [wiki](https://github.com/archzfs/archzfs/wiki) for user
documentation. The [architecture](docs/architecture.md) document describes the
current build and release system, while the [roadmap](docs/roadmap.md) records
maintainer direction and unresolved policy. Contributors and automation agents
should also read the repository's [agent instructions](AGENTS.md).
